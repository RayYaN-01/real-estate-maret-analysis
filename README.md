# Hyderabad Real Estate Market Analysis

## Project Overview

This project analyzes Hyderabad's real estate market using Python, MySQL, and Power BI. The workflow includes data cleaning, feature engineering, SQL-based analytics, and interactive dashboard development for business insights and investment analysis.

---

## Tech Stack

- Python
- Pandas
- NumPy
- MySQL
- Power BI
- Jupyter Notebook

---

## Key Features

- Data cleaning and preprocessing
- Feature engineering
- Property price segmentation
- Price per sqft analysis
- SQL analytical queries
- Interactive Power BI dashboards
- Investment and luxury property insights

---

## Dashboard Pages

### 1. Executive Market Overview
- Property distribution analysis
- Bedroom-wise pricing trends
- City-wise market comparison
- Price segment distribution

### 2. Investment & Property Analytics
- Property size vs pricing analysis
- Investment distribution by segment
- Avg area by bedroom type
- Price per sqft analysis

### 3. High Value Property Analysis
- Luxury property insights
- Premium property table
- Property value treemap
- Luxury pricing analysis

## Key Insights

- Independent houses/villas have the highest average prices.
- Ultra luxury properties dominate high-value investments.
- Property prices increase significantly with bedroom count.
- Hyderabad properties have higher average prices than Secunderabad.
- Luxury segments show the highest price per sqft values.
